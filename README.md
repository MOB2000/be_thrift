# be_thrift

A Flutter project to manage your income and expenses, safe and secure

## Features

- Beautiful minimal UI
- Create Account with Google
- Use app Anonymously
- View Balance
- Add income & expense
- Add Categories
- Change App Color
- Select language
- Select currency
- change username & email
- Delete Account
